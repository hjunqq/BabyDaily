# BabyDaily UI/UX 设计规范（优雅温柔版）

## 1. 设计理念
- 核心气质：温柔、克制、亲密、轻奢（Soft Luxury）。
- 目标感受：高级育儿手账 + 清爽工具感，拒绝“糖果感过载”。
- 关键词：温暖、清晰、亲子、低刺激。

## 2. 视觉识别
### 2.1 色板（Sakura Blush）
- Primary / Rose Blush：`#F3B6C2`
- Text / Cocoa：`#4A342E`
- Background / Cream：`#FFF6F2`
- Secondary / Sand：`#F7EFEB`
- Accent / Sage：`#BFD9C6`
- Accent / Mist Blue：`#CFE9F2`
- Line / Divider：`#E8DCD6`
- Danger / Dusty Coral：`#E58C86`

### 2.2 字体与排版
- 中文标题：`思源宋体 SC` / `Noto Serif SC`
- 正文字体：`HarmonyOS Sans SC` / `Noto Sans SC`
- 英文字体：`Fraunces`（标题）+ `Manrope`（正文）
- 字阶：H1 24/32、H2 20/26、H3 18/24、Body 15/16、Caption 12/13
- 行高：1.5–1.7；段落间距明显。

### 2.3 形态与阴影
- 圆角：大卡片 18–20px、小卡片 14–16px、输入框 12px、按钮 14–16px。
- 阴影：单层柔和 `(0,10,30, rgba(66,41,33,0.12))`。
- 玻璃效果：最多 1–2 处（头像或主按钮）。

### 2.4 图标与插画
- 图标：线性为主，统一线宽 1.6px；避免重玻璃与复杂渐变。
- 插画：轻线稿或柔和块面，控制装饰密度。

## 3. 布局与间距
- 栅格：8pt 基准；卡片内边距 16–20px。
- 点击/触控：最小 44px。
- 内容区留白 > 装饰元素。

## 4. 组件规范
### 4.1 按钮
- 主按钮：主色底 + 深色或白字；按压缩放 0.96。
- 次按钮：浅底 + 边框；悬停加深边框。
- 状态：default / hover / press / disabled / loading。

### 4.2 卡片
- KPI 卡、列表卡、相册卡统一圆角与阴影。
- 卡片标题与主数字层级明确。

### 4.3 表单
- 输入框：圆角 12px + 1px 线框。
- 焦点态：描边高亮 1–2px。
- 错误态：红色描边 + 文案提示。

### 4.4 标签与徽章
- 轻色底 + 深色字；用于状态与分类。

## 5. 关键页面规范
### 5.1 移动端 Home
- 顶部：头像玻璃卡 + 宝宝名 + 日期。
- 今日关键总览：3 卡横排（奶量/尿布/睡眠）。
- 最近记录：3–5 条 + 查看全部。
- 主操作：底部浮动“+ 记录”。

### 5.2 移动端 Record
- 记录类型 Tab + 表单输入。
- 快捷量表 chips。
- 提交按钮固定底部。

### 5.3 移动端 Statistics
- 日期范围切换 + 折线/柱状图。
- 底部概览卡：均值、峰值、趋势。

### 5.4 OOTD 相册
- 月历 + 瀑布流。
- 拍照主按钮固定底部。

### 5.5 Web Dashboard
- 顶部栏 + 侧边栏 + KPI 卡 + 趋势图 + 表格。
- 表格：表头底纹、行高 ≥ 44px、操作列有 tooltip。

## 6. 动效规范
- 页面进入：轻淡入 + 上移 6–10px。
- 卡片 hover：y 轻浮起，阴影增强 10%。
- 主按钮：按压缩放 0.96。

## 7. 可用性与对比度
- 对比度遵守 WCAG AA。
- 文案简洁、清晰、避免卡通过度。

## 8. A/B 方向
- A 版：保留可爱风但去装饰。
- B 版：轻奢清爽为主方向（推荐）。
