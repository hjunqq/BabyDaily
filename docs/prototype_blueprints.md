# BabyDaily 高保真原型文字蓝图 vNext（优雅温柔版）

## 核心定位
- 关键词：温柔、克制、亲密、轻奢（Soft Luxury）。
- 目标感受：像“婴儿房的高级香氛”，不是糖果店；视觉轻、层次清、信息优先。
- 主题承诺：可爱但不幼稚，甜而不腻，保留亲子温度。

## 视觉基线（必须遵守）
### 色彩（建议变量）
- 主色 Rose Blush：`#F3B6C2`
- 主文字 Cocoa：`#4A342E`
- 背景 Cream：`#FFF6F2`
- 次级背景 Sand：`#F7EFEB`
- 点缀 Sage：`#BFD9C6`
- 点缀 Mist Blue：`#CFE9F2`
- 分隔线 Line：`#E8DCD6`
- 危险色 Dusty Coral：`#E58C86`

### 字体与层级
- 标题：中文推荐 `思源宋体 SC` / `Noto Serif SC`；英文 `Fraunces`。
- 正文：中文推荐 `HarmonyOS Sans SC` / `Noto Sans SC`；英文 `Manrope`。
- 字阶（移动/桌面）：
  - H1 24/32 700
  - H2 20/26 600
  - H3 18/24 600
  - Body 15/16 500
  - Caption 12/13 500
- 行高：1.5–1.7，尽量留白。

### 形态与阴影
- 圆角：大卡片 18–20px；小卡片 14–16px；输入框 12px；按钮 14–16px。
- 阴影：柔和一层 `(0,10,30, rgba(66,41,33,0.12))`，避免多层高光。
- 玻璃：仅允许 1–2 处（头像或主按钮），且透明度低、边缘清爽。

### 图标与插画
- 图标：线性为主，少量面填；统一 1.6px 线宽。
- 插画：轻线稿或柔和块面，比例克制。

### 触控与可用性
- 触控区 ≥ 44px。
- 文字与背景对比度符合 WCAG AA。
- 输入、按钮、卡片均有 hover/press/disabled/loading。

## 移动端 Home vNext（Sakura Baby Tracker）
### 布局（自上而下）
1) 顶部区
- 左侧：玻璃头像 80–88px，右侧显示宝宝名、今天日期。
- 标题：H1 + 副标题（今日状态一句话）。

2) 今日关键总览（3 卡横排）
- 卡片宽 110–124px，高 100–116px，圆角 18px。
- 指标：奶量、尿布、睡眠。
- 结构：标题 + 主数值 + 微型进度或次级文案。

3) 今日时间轴（最近 3–5 条）
- 列表卡：浅底色、轻分隔线。
- 左：时间与类型；右：数值或状态标签。
- 末尾：次要按钮“查看全部”。

4) 主操作
- 悬浮主按钮“+ 记录” 58–64px 高，主色底，按压缩放 0.96。
- 次要操作收敛在半屏 Action Sheet。

### 视觉/动效
- 背景：柔和渐变 + 轻颗粒纸感纹理（极弱）。
- 花瓣点缀：≤6，远离正文，透明度低。
- 卡片 hover 轻浮起，阴影加强 10%。

### 空态/错误态
- 空态：小插画 + 主按钮引导。
- 错误态：顶部轻提示条，支持重试。

## 移动端 Record（快速记录）
- 结构：顶部选择记录类型（奶量/尿布/睡眠/辅食/其他）+ 主表单。
- 快捷量表：常用数值 chips（60/90/120/150 ml）。
- 时间选择：底部弹层 + 可编辑文本。
- 提交按钮固定底部，点击反馈明确。

---
## 移动端 Statistics（趋势）
- 顶部：日期范围切换（7 天/30 天/自定义）。
- 中部：折线图（奶量/辅食）+ 柱状（尿布/睡眠）。
- 底部：概览卡（均值、峰值、对比上周）。
- 坐标与单位清晰，颜色对比明确。

## OOTD 宝宝穿搭相册
- 入口：Home 快捷卡 + Tabbar。
- 月历视图：方形卡片，缩略图 + 小标签。
- 瀑布流：照片卡片圆角 16px，阴影轻，标签 chip（风格/场合）。
- 拍照按钮：圆形主按钮“📷 记录今日穿搭”。

## Web Dashboard vNext（Little Blossom Tracker）
### 布局（1440px 基准）
1) 顶部栏
- 左：Logo + 标题；中：搜索；右：通知、语言、头像。

2) 侧栏
- 简洁线性图标；选中态：左侧高亮条 + 轻底色。

3) KPI 区
- 3–4 卡，宽 260–300px，高 150–170px。
- 标题 14px 600；数值 30px 700；副文案 12px。

4) 趋势图
- 7 天/30 天切换；双色区分奶量/辅食。
- 必须有坐标、网格线、单位。

5) 近期活动表格
- 表头底纹，行高 ≥ 44px；操作列按钮 + tooltip。

### 视觉要求
- 背景：浅米/浅灰；主色仅用于强调。
- 表格与图表优先信息性，装饰克制。
